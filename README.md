# IoTSecurityMQTTAMQP
This project demonstrates the evaluation and implementation of Traditional IoT Messaging Protocols (MQTT, AMQP) over TCP/TLS and QUIC along with a rate-limited IPS Setup. Consists of Publisher, Broker and Subscriber intended to be used in 3 VMs across a NAT. Two GUIs for Publisher and Subscribers and one CLI for Broker to see properties.
